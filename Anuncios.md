# Anuncios personalizados con Ad Inserter y HTML generado por IA
Para mostrar anuncios personalizados dentro del sitio web, se utilizó el plugin Ad Inserter, que permite insertar código HTML en diferentes zonas de la página. El contenido de los anuncios se generó mediante IA y se pegó en los bloques del plugin.
### Funciones principales de Ad Inserter:
1. Inserción de anuncios en cabecera, pie, sidebar o dentro del contenido.
2. Soporte para HTML, CSS y JavaScript.
3. Varios bloques para gestionar diferentes anuncios.
4. Condiciones de visualización (solo en entradas, solo en páginas, etc.).
### Pasos a seguir:
1. Ir a Plugins → Añadir nuevo.
2. Buscar Ad Inserter.
3. Pulsar Instalar y luego Activar.
4. Ir al menú Ad Inserter en el panel lateral.
5. Seleccionar un bloque (por ejemplo, Block 1).
6. Pegar el código HTML generado por IA.
7. Configurar la ubicación del anuncio (antes del contenido, después, entre párrafos, widgets, etc.).
8. Guardar los cambios.
9. Comprobar en la web que el anuncio se muestra correctamente.
