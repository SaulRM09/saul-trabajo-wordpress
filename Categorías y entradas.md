# Categorías
Las categorías permiten organizar el contenido del sitio web y facilitar la navegación del usuario. Son especialmente útiles en blogs o páginas con muchas publicaciones, ya que agrupan las entradas por temática y mejoran la estructura del sitio.
### Pasos a seguir:
1. Acceder al panel de administración de WordPress.
2. En el menú lateral, seleccionar Entradas → Categorías.
3. Rellenar los campos necesarios: Nombre de la categoría, Slug (opcional), que será la URL amigable, Categoría superior si deseas crear subcategorías, Descripción (opcional).
4. Pulsar Añadir nueva categoría.
5. Repetir el proceso para crear todas las categorías necesarias.

# Creación de una entrada
Las entradas son el contenido principal de un blog o sección de noticias. Cada entrada puede asignarse a una o varias categorías para mantener el sitio organizado.
### Pasos a seguir:
1. Ir a Entradas → Añadir nueva.
2. Escribir el título de la entrada.
3. Añadir el contenido en el editor (texto, imágenes, vídeos, etc.).
4. En la barra lateral, seleccionar la categoría correspondiente.
5. Añadir etiquetas si deseas mejorar la organización.
6. Insertar imágenes o archivos desde Añadir objeto.
7. Revisar la entrada con Vista previa.
8. Publicar la entrada pulsando Publicar.
