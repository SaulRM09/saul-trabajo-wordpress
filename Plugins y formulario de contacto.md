# Plugins esenciales utilizados
En este proyecto se han utilizado varios plugins para ampliar las funcionalidades del sitio:
1. Plugin de formularios (para el formulario de contacto).
2. Plugin de seguridad.
3. Plugin de SEO.
4. Plugin de anuncios (Ad Inserter).
5. Plugin de traducción.
6. Plugin para foto de perfil de usuarios.
### Instalación de un plugin:
1. Ir a Plugins → Añadir nuevo.
2. Escribir el nombre del plugin en el buscador.
3. Pulsar Instalar ahora.
4. Hacer clic en Activar.
5. Configurarlo desde su menú correspondiente.

# Formulario de contacto
Para permitir que los usuarios se comuniquen con el sitio, se creó un formulario de contacto mediante un plugin.
### Pasos a seguir:
1. Instalar un plugin de formularios (por ejemplo, Contact Form 7 o WPForms).
2. Acceder al menú del plugin y crear un nuevo formulario.
3. Añadir campos como nombre, correo y mensaje.
4. Guardar el formulario.
5. Copiar el shortcode generado.
6. Ir a la página de Contacto.
7. Pegar el shortcode en el contenido.
8. Actualizar o publicar la página.
9. Probar el formulario desde la web.
