# Colocación del tema
En WordPress ya vienen incluidos miles de temas prediseñados que podemos utilizar para construir nuestra página web. Estos temas ofrecen diferentes estilos, estructuras y diseños que permiten adaptar la apariencia del sitio sin necesidad de programar desde cero. Gracias a ellos, es posible crear páginas modernas y funcionales simplemente seleccionando el tema que mejor se ajuste a las necesidades del proyecto y personalizándolo mediante sus opciones integradas.
### Pasos a seguir:
Para añadir un tema a nuestro sitio de WordPress, debemos seguir los siguientes pasos, que se muestran también en las imágenes adjuntas:
1. Acceder al panel de administración de WordPress.
2. Ir al menú lateral y seleccionar **Apariencia → Temas**.
3. Hacer clic en **Añadir nuevo**.
4. Buscar el tema deseado o explorar los destacados.
5. Pulsar **Instalar**.
6. Hacer clic en **Activar** para aplicarlo al sitio.
7. Personalizar el tema desde **Apariencia → Personalizar**.

# Creación de usuarios
WordPress permite gestionar diferentes tipos de usuarios dentro del sitio web, cada uno con permisos y funciones específicas. Esto resulta útil cuando se necesita que varias personas colaboren en la administración, redacción o mantenimiento del sitio. Gracias a este sistema de roles, es posible controlar quién puede editar contenido, instalar plugins, moderar comentarios o realizar cambios en la configuración.
### Pasos a seguir:
1. Acceder al panel de administración de WordPress.
2. En el menú lateral, seleccionar la opción Usuarios → Añadir nuevo.
3. Rellenar los campos obligatorios: Nombre de usuario, Correo electrónico, Nombre y apellidos (opcional), Sitio web (opcional).
4. Establecer una contraseña para el nuevo usuario o generar una automáticamente.
5. Activar o desactivar la opción de Enviar aviso al usuario, según si deseas que reciba un correo con sus datos de acceso.
6. Seleccionar el rol del usuario según los permisos que necesite: Suscriptor, Colaborador, Autor, Editor, Administrador.
7. Hacer clic en Añadir nuevo usuario para guardar los cambios.
